# CircuLib 📚
A Library Management Platform designed to simplify book tracking, requests, and borrowing for students and librarians.
This project is being developed as a capstone project.

## 🚀 Features
- Search and request books
- Manage borrowing and due dates
- Track usage and analytics
- Automated notifications (UI planned)
- Role-based access (UI planned)

- ## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
(Backend to be added)

## 🔮 Future Improvements
- Backend integration
- Authentication system
- Real-time notifications
- Dashboard for admins/librarians
- Responsive improvements
